# HoneyGen - HGT
This folder contains the following:
1. The code for the HoneyGen HGT Model, also available at the following [bitbucket link](https://bitbucket.org/srecgrp/honeygen-generating-honeywords-using-representation-learning/src/master/);
2. My modular implementation for the HoneyGen HGT Model, coupled with a way to substitute embedding models.
