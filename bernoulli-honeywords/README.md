# Bernoulli Honeywords

This folder contains the implementation and experiments for Bernoulli Honeywords. The integration with the honeychecker design can be found in the ```honeychecker``` folder. The integration with the Amnesia local design can be found in the ```amnesia/local``` folder. The ```output``` folder shows all the plots generated by the experiments.

## How to run the experiments

To run the experiments, first make sure you are in the right folder.

```
cd bernoulli-honeywords/implementation
```

Also make sure that you have python installed on your machine.
Then run:

```
pip install -r requirements.txt
```

In order to run the experiments, the hashmob dataset with counts is needed. It can be found [here](https://drive.google.com/drive/folders/1sTQCVNtSV7Y2X6jLnhCrFEUcs3K7HKJq?usp=sharing) (the file ```hashmob_counts.txt```). Once you have downloaded the file, add it to the ```data``` folder. 

To run the experiments, run:

```
python -m experiments.run_experiments
```
